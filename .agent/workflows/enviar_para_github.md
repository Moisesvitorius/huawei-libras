---
description: Envia alterações para o GitHub solicitando descrição e confirmando mudanças
---

// turbo-all

1. Execute `git status` no terminal para visualizar os arquivos modificados.
2. Pergunte ao usuário: "Quais foram as mudanças e qual a descrição do commit?" (Caso o usuário não tenha detalhado ainda).
3. Aguarde a resposta do usuário.
4. Execute os comandos no terminal:
   - `git add .`
   - `git commit -m "MENSAGEM_DO_USUARIO"`
   - `git push`
5. Confirme o sucesso da operação.
